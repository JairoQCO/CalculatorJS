# Calculadora Web Básica

Calculadora simple desarrollada únicamente con **HTML, CSS y JavaScript**, sin librerías ni frameworks externos.

## Funcionalidades
- Operaciones básicas: suma, resta, multiplicación y división
- Soporte para decimales
- Funciones: `AC`, `DEL`, porcentaje (`%`) y cambio de signo (`±`)
- Manejo de errores (ej. división entre 0)
- Soporte de teclado

## Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)

## Uso
1. Clona o descarga el repositorio
2. Abre el archivo `index.html` en tu navegador
3. Usa la calculadora desde la interfaz o el teclado

## Propósito
Proyecto de práctica para validar:
- Uso de Git y GitHub por SSH
- Manipulación del DOM
- Lógica básica en JavaScript
